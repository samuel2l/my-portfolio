# Samuel Adams — Developer Portfolio

React portfolio. Deploy on [Render](https://render.com) as a **Static Site**.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Render

1. Push this repo to GitHub.
2. In Render: **New** → **Static Site**.
3. Connect your repo.
4. **Build command:** `npm run build`
5. **Publish directory:** `dist`
6. Deploy.

Your site will be served from the `dist` folder after each build.
